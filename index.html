About Me

Personal Project

Dota100

Voted "Best Overall" & "Most-Shippable App" in my cohort. I was able to integrate a 3rd party api and create a user interface where users could link their Game Account ID# and see relative information. Including being able to follow other registered users and compare data with one another.

Being able to include other packages like Chart.js & D3 were a challenge when dealing with a 3rd party api. Reformatting api information and converting it to both different npm package correcting formatting, and the way users want to view it, was a fun an interesting learning challenge.

React | Redux | HTML5 | CSS3 | NodeJS | Axios | Express | Express Session | Massive | PostgreSQL | Bcrypt | Chart.js | D3 | Heroku

Mountain Chess
